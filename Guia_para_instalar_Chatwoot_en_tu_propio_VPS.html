🚀 Guía de Instalación de Chatwoot v4.1.0 con Docker y Traefik

📋 Requisitos Previos

  • VPS con Ubuntu 24.04 LTS (Hostinger o similar)
  • Docker y Docker Compose instalados
  • n8n y Traefik ya configurados y funcionando
  • Mínimo 4GB RAM y 50GB disco

Paso 1: Crear la Red Externa

Esta red permite que Traefik, n8n y Chatwoot se comuniquen. Al ser externa, no se destruye cuando actualizas servicios.

docker network create traefik_network
Nota: Si ya existe, verás un error que puedes ignorar.

Paso 2: Modificar el archivo de Traefik y n8n

Abre tu archivo:

nano ~/docker-compose.yml

Realiza estos 4 cambios:

2.1 En la sección command de Traefik, añade esta línea:

      - "--providers.docker.network=traefik_network"

2.2 Al final del servicio traefik, añade:

    networks:
      - traefik_network

2.3 Al final del servicio n8n, añade:

    networks:
      - traefik_network

2.4 Al final del archivo, después de volumes, añade:

networks:
  traefik_network:
    external: true

Guarda: Ctrl + O, Enter, Ctrl + X

Paso 3: Reiniciar Traefik y n8n

cd ~
docker compose down
docker compose up -d

Verifica que funcionan:

docker ps

Deberías ver root-traefik-1 y root-n8n-1 en estado Up.

Paso 4: Crear la carpeta de Chatwoot

mkdir ~/chatwoot

Paso 5: Crear el archivo .env de Chatwoot

nano ~/chatwoot/.env

Pega el siguiente contenido:

# Dominio donde accederás a Chatwoot
FRONTEND_URL=https://chatwoot.TU_DOMINIO

# Entorno de producción
RAILS_ENV=production

# Permitir crear cuentas desde la URL
ENABLE_ACCOUNT_SIGNUP=true

# Claves secretas (GENERA NUEVAS - ver abajo)
SECRET_KEY_BASE=PEGA_AQUI_TU_CLAVE_64
POSTGRES_PASSWORD=PEGA_AQUI_TU_CLAVE_32

# Base de datos PostgreSQL
POSTGRES_HOST=postgres
POSTGRES_PORT=5432
POSTGRES_DATABASE=chatwoot_production
POSTGRES_USERNAME=chatwoot

# Redis
REDIS_URL=redis://redis:6379

# Almacenamiento local (para archivos, audios, imÃ¡genes)
ACTIVE_STORAGE_SERVICE=local

Guarda: Ctrl + O, Enter, Ctrl + X

⚠️ Genera claves secretas únicas: 
openssl rand -hex 64

Copia el resultado y pégalo en SECRET_KEY_BASE

openssl rand -hex 32

Copia el resultado y pégalo en POSTGRES_PASSWORD

Paso 6: Crear el docker-compose.yml de Chatwoot

nano ~/chatwoot/docker-compose.yml

Pega el siguiente contenido (cambia TU_DOMINIO):

services:
  postgres:
    image: ankane/pgvector
    restart: always
    volumes:
      - postgres_data:/var/lib/postgresql/data
    environment:
      - POSTGRES_USER=${POSTGRES_USERNAME}
      - POSTGRES_PASSWORD=${POSTGRES_PASSWORD}
      - POSTGRES_DB=${POSTGRES_DATABASE}

  redis:
    image: redis:7-alpine
    restart: always
    volumes:
      - redis_data:/data

  chatwoot:
    image: chatwoot/chatwoot:v4.1.0
    restart: always
    depends_on:
      - postgres
      - redis
    env_file:
      - .env
    volumes:
      - chatwoot_storage:/app/storage
    command: bundle exec rails s -p 3000 -b 0.0.0.0
    labels:
      - traefik.enable=true
      - traefik.http.routers.chatwoot.rule=Host(`chatwoot.TU_DOMINIO`)
      - traefik.http.routers.chatwoot.tls=true
      - traefik.http.routers.chatwoot.entrypoints=websecure
      - traefik.http.routers.chatwoot.tls.certresolver=mytlschallenge
      - traefik.http.services.chatwoot.loadbalancer.server.port=3000
    networks:
      - default
      - traefik_network

  sidekiq:
    image: chatwoot/chatwoot:v4.1.0
    restart: always
    depends_on:
      - postgres
      - redis
    env_file:
      - .env
    volumes:
      - chatwoot_storage:/app/storage
    command: bundle exec sidekiq -C config/sidekiq.yml

volumes:
  postgres_data:
  redis_data:
  chatwoot_storage:

networks:
  traefik_network:
    external: true

Guarda: Ctrl + O, Enter, Ctrl + X

💾 Almacenamiento persistente: Los volúmenes chatwoot_storage garantizan que los archivos (audios, imágenes, documentos) enviados en las conversaciones se guarden permanentemente, incluso si reinicias o actualizas los contenedores. La variable ACTIVE_STORAGE_SERVICE=local le indica a Rails que use el sistema de archivos local en lugar de servicios en la nube como S3.

Paso 7: Levantar Chatwoot

cd ~/chatwoot
docker compose up -d

Espera 30 segundos:

sleep 30

Verifica que estén corriendo:

docker compose ps

Deberías ver 4 contenedores en estado Up:

  • chatwoot-postgres-1
  • chatwoot-redis-1
  • chatwoot-chatwoot-1
  • chatwoot-sidekiq-1

Paso 8: Configurar la Base de Datos

Ejecuta estos comandos en orden (desde ~/chatwoot):

8.1 Crear la base de datos

docker compose run --rm chatwoot bundle exec rails db:create

8.2 Ejecutar las migraciones

docker compose run --rm chatwoot bundle exec rails db:migrate

Este proceso puede tardar 2-3 minutos.

8.3 Cargar datos iniciales

docker compose run --rm chatwoot bundle exec rails db:seed

Paso 9: Verificar la conexión de red

docker network inspect traefik_network --format '{{range .Containers}}{{.Name}} {{end}}'

Deberías ver: root-traefik-1, root-n8n-1 y chatwoot-chatwoot-1

Paso 10: Acceder a Chatwoot

Abre tu navegador y ve a:

🌐 https://chatwoot.tudominio.com

Crear tu cuenta

  1. Haz clic en "Create a new account"
  2. Completa: nombre, email y contraseña
  3. Haz clic en "Create account"

🛠️ Gestión y Actualizaciones

✅ Ventaja de la red externa: Puedes actualizar n8n, Traefik o Chatwoot de forma independiente sin que se desconecten entre sí.

Actualizar n8n y Traefik

cd ~
docker compose pull
docker compose down
docker compose up -d

Chatwoot seguirá funcionando.

Actualizar Chatwoot

cd ~/chatwoot
nano docker-compose.yml
# Cambiar la versión: image: chatwoot/chatwoot:v4.X.X

docker compose pull
docker compose down
docker compose up -d
sleep 30
docker compose run --rm chatwoot bundle exec rails db:migrate

Ver logs

# Logs de Chatwoot
cd ~/chatwoot
docker compose logs chatwoot -f

# Logs de n8n
cd ~
docker compose logs n8n -f

🔒 Seguridad Post-Instalación

Desactiva el registro público después de crear tu cuenta:

nano ~/chatwoot/.env
# Cambiar: ENABLE_ACCOUNT_SIGNUP=false

cd ~/chatwoot
docker compose restart chatwoot

🗑️ Desinstalar Chatwoot

Opción 1: Mantener datos

cd ~/chatwoot
docker compose down

Opción 2: Eliminar todo

cd ~/chatwoot
docker compose down -v
cd ~
rm -rf ~/chatwoot
